## Checklist for using the bot
- **Did you read the readme in the repo?**
  - [Readme link](../README.md#before-you-start)
- **Is python installed?**
  - Check with:    `py --version`
- **Is pip installed?**
  - Check with:    `py -m pip --version`
- **Are you in the right folder?**
  - Check with:    `dir`
  - Are `main.py` and `requirements.txt` in the list?
- **Is the game on correct graphic settings (standard)?**
- **Did you turn off all confirmations? (rest, infirmary etc)**
- **If you're using an Android Emulator:**
  - Did you set your screen size to (`800x1080`) ?
  - Did you set your DPI to 240?
  - Did enable use_adb for native emulator use and set your emulator's IP:Port combination in the web UI?
- **Have you updated the bot?**
  -  Use "git pull" command or GitHub Desktop and click on the button that says pull (or re-download zip)
- **Is the game running on your primary monitor? (for steam and non ADB emulator only)**
- **Are you using any mods?**  (mods are not supported)
- **Are you running the game as administrator?** (if so, either don't or run the bot as admin too)

## Checklist for reporting a bug / asking for help, so that we have more of an understanding of your problem:
- **At what point do you need help?** (Installation / Running / Crashing)
  - Describe it as best you can.
- **Are you using Steam or Android Emulator?**
  - If emulator, which one?
- **Have you copied or screenshotted your logs when the error happened?**
  - We can use these to identify your problem faster and help you faster.
- **What is the version of the bot?**
  - Should be shown in the UI next to Uma Auto Train title, if there's no version, branch name is helpful.
